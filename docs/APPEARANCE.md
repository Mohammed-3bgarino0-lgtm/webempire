# Visual Identity Engine

The public site has no mandatory fixed brand color.

Admin path:

`/admin/appearance`

## Theme presets

- Minimal Clean
- AI SaaS
- Dark Premium
- Glass UI
- Neo Bold
- Corporate

A preset is only a starting style. Individual settings remain independent.

## Adjustable values

- Primary color
- Accent color
- Background color
- Surface color
- Text color
- Dark background
- Dark surface
- Dark text
- Header style
- Hero style
- Card style
- Border radius
- UI density
- Desktop columns
- Tablet columns
- Mobile columns
- Default color mode
- Font preset

Public pages receive the settings through CSS variables and data attributes.

## Visitor color mode

The public header includes separate Light, Dark and System buttons. The visitor choice is stored in browser local storage and applied to the public shell.

## Localized site identity

Each active locale can have its own:

- Site name
- English name
- Tagline
- Home SEO title
- Home SEO description
